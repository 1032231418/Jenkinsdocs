## [第二章 Docker基本操作](chapter/第二章-Docker基本操作.md)

- search 搜索镜像
- pull 下载镜像
- images 列出镜像
- run 运行容器
- ps 查看容器列表
- 控制容器
- attach 连接容器
- exec 外部运行容器命令
- rm 删除容器
- rmi 删除镜像
- history 镜像历史
- cp 复制文件
- commit 提交容器修改
- diff 检查容器文件的变动
- inspect 查看详细信息
